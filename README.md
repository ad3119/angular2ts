angular2

Installing :

1.Make sure you have Node and Git Installed  in your machine.
2. Clone the repo and get into the directory where index.html lies through command prmopt.
3. type in below codes for typescript, node dependencies and bower dependencies to add to the app directory.
	npm install
	bower install
4. Start the node-lite server for typescript to work and generate the *.js files when a *.ts is changed in the app directory.
